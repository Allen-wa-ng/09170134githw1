# Merge The Number!
## Keybinding
- M - mute the sound
- V - vertical super power
- H - horizontal super power
- DOWN_KEY - move to the bottom
- LEFT_KEY - move to the left
- RIGHT_KEY - move to the right
- ENTER_KEY/SPACE - pause the game
## Prerequirements
- python
- pygame
- pip
## Installation
```pip3 install pygame```
